# Calendar

**/!\ THIS IS AN ALPHA VERSION, WIP**

Calendar is a TUI app to manage your markdown-calendar.

## Features

- [x] Month view
- [x] Week view
- [ ] Day view
- [x] Year view
- [!] Add events
- [ ] Edit events
- [ ] Remove events
